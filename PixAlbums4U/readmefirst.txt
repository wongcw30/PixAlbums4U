Note that PixAlbums4U must be in the directory C:\wamp\www only.

Please config your server before run this web application.

Edit the 'php.ini'
----------------------------------------------------------
max_execution_time = 300     
post_max_size = 50M
upload_max_filesize = 50M
max_file_uploads = 100
memory_limit = 256M
----------------------------------------------------------

Enjoy.
